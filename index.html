Projects

XinMalwareScanner – ML-Based Malware Detection System

XinMalwareScanner is a file-based malware detection system designed to reduce human intervention by leveraging an ensemble of machine learning models. It analyzes file behavior and characteristics to classify files as Safe, Infected, or Unknown.

Key features include support for file/folder scanning, auto-generated reports, dark mode UI, model upload/reset capabilities, and an intelligent feedback loop that learns from user-reported false positives or negatives.

Weather – Web-Based Weather Checking

A responsive weather forecast app built with React. It fetches real-time weather data from OpenWeatherMap API and visually adapts the background based on the weather condition (e.g., clear, clouds, rain). Includes current weather, location-based forecast, and dynamic UI interactions.
